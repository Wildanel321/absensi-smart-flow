┌─────────────────────────────────────────────────────────────────────────┐
│                           NOTES & TIPS                                   │
└─────────────────────────────────────────────────────────────────────────┘

   ✓ MFRC522 uses SPI communication
   ✓ LCD uses I2C communication  
   ✓ SDA pin is shared between MFRC522 (SPI) and LCD (I2C) - this is OK!
   ✓ Always use 3.3V for MFRC522, NOT 5V
   ✓ Use 220Ω resistors for LEDs
   ✓ Double-check all GND connections
   ✓ Keep wires short for stable SPI communication
   ✓ Test RFID reader first before adding other components
